TopDownSimpleService

Overview

TopDownSimpleService exposes operations that provide a user access

 

external market data given some search ﬁlters and for this data to be

 

downloaded as a CSV ﬁle.

End point address

http://localhost:8080/TopDownSimpleService/TopDownSimpleService?

 

wsdl

Operations

importMarketData, downloadFile

 

 

 

ImportMaketData

Description

Imports market data from the external source Yahoo Finance by specifying

 

some search ﬁlters. The data is then formatted and stored locally on the

 

server as a CSV ﬁle at the location ${catalina.home}/webapps/ROOT/

 

marketdata.

Inputs

sec - The stock code of the ﬁnancial institution. Expected Format: 3 Letters

 

(e.g., BHP, TLS, AGL)

 

startDate - Start date to include in search. Format: DD-MM-YYYY (e.g.

 

01-01-2010)

 

startDate - End date to include in search. Format: DD-MM-YYYY (e.g.

 

01-01-2010)

 

dataSourceURL - Not being used.

Outputs

eventSetId - The unique identiﬁer for this imported data.

 

 

Faults

importMarketFault

 

ERR_SEC - invalid stock code

 

ERR_DATE - invalid date format

 

ERR_DATE_RANGE - end date is before start date

 

ERR_SERVICE - internal service error

the supplied unique ID. The ﬁle is served from the local storage location $

 

 

{catalina.home}/webapps/ROOT/marketdata.

 

 

 

Inputs

eventSetId - The unique identiﬁer for the imported data.

 

 

 

 

 

 

 

 

Outputs

dataURL - The URL to access the CSV ﬁle.

 

 

 

 

 

 

 

 

Faults

downloadFileFault

 

 

 

 

ERR_ID - unknown id

 

 

 

 

ERR_SERVICE - internal service error

 

 

 

MarketDataUtilService

Overview

MarketDataUtilService processes and manipulates already imported market

 

data. This requires the TopDownSimpleSerivce importMarketData to be

 

executed before hand.

End point address

http://localhost:8080/MarketDataUtilService/MarketDataUtilService?wsdl

 

 

Operations

summariseMarketData, convertMarketData, visualiseMarketData

 

 

 

SummariseMarketData

Description

Outputs a summary of a existing imported market data by reading the CSV

 

ﬁle stored at ${catalina.home}/webapps/ROOT/marketdata and extracting

 

the meta data from it.

Inputs

eventSetId - The unique identiﬁer for the imported data.

 

 

Outputs

eventSetId - The unique identiﬁer for this imported data.

 

sec - The stock code for this imported data.

 

startDate - The starting date referenced in the imported data.

 

endDate - The end date referenced in the imported data.

 

currencyCode - The currency referenced in the imported data.

 

ﬁleSize - The size of the imported data in KB.

Faults

summariseMarketDataFault

 

ERR_ID - unknown id

 

ERR_SERVICE - internal service error

ConvertMarketData

 

 

 

 

 

 

 

Description

Converts an existing imported market data to a different currency. It uses the

 

 

 

external source XE Currency Tables to ﬁnd the exchange rate for the

 

 

targeted currency and creates a new CSV ﬁle with the converted market

 

 

values. This new ﬁle is stored at the local location ${catalina.home}/

 

 

webapps/ROOT/marketdata.

 

 

 

Inputs

eventSetId - The unique identiﬁer for the imported data to convert.

 

 

 

 

targetCurrency - The country currency code to which price data are to be

 

 

 

converted. Format: three letter currency code (e.g., USD, EUR)

 

 

 

 

targetDate - The preferred date to choose the conversion rate. Format:

 

 

 

YYYY-MM-DD

 

 

 

Outputs

eventSetId - The new unique identiﬁer for the converted market data.

 

 

 

 

 

 

Faults

convertMarketDataFault

 

 

 

 

ERR_ID - unknown id

 

 

 

 

ERR_DATE - invalid date format

 

 

 

 

ERR_CURR - invalid currency format

 

 

 

 

ERR_SERVICE - internal service error

 

 

 

 

VisualiseMarketData

Description

Creates a HTML ﬁle that formats the imported market data for viewing in a

 

web browser. The HTML ﬁle is generated and stored at the local location $

 

{catalina.home}/webapps/ROOT/marketdata.

Inputs

eventSetId - The unique identiﬁer for the imported data to visualise.

 

 

Outputs

dataURL - The URL to access the HTML ﬁle.

 

 

Faults

visualiseMarketDataFault

 

ERR_ID - unknown id

 

ERR_SERVICE - internal service error
